# Telecom Data Platform

This project demonstrates an end-to-end data engineering pipeline combining:

- Batch processing
- Real-time streaming
- ELT transformations

## Tech Stack

- Python
- PostgreSQL
- Kafka
- dbt
- Airflow
- Docker

## Project Structure

- ingestion/ → data ingestion (batch + streaming)
- dbt/ → transformations
- airflow/ → orchestration
- docker/ → infrastructure
